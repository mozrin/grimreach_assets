# Grimreach Assets

This project stores non-code, non-runtime, non-generated visual and branding assets for the entire Grimreach workspace.

## Structure

- `branding/`: Logos, marks, icons.
- `ui/`: UI elements (future).
- `world/`: World-space visual assets (future).
- `temp/`: Scratch space.

## Contents

- `branding/grimreach_gemini_logo.png`: Logo from Gemini
- `branding/grimreach_copilot_logo.png`: Logo from Copilot
